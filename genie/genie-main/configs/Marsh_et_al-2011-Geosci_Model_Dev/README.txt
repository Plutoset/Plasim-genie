These configuration files can be used to produce the model results presented in the publication entitled 'Incorporation of the C-GOLDSTEIN efficient climate model into the GENIE framework: "eb_go_gs" configurations of GENIE' (Marsh, R., MÃ¼ller, S. A., Yool, A., and Edwards, N. R., 'Incorporation of the C-GOLDSTEIN efficient climate model into the GENIE framework: "eb_go_gs" configurations of GENIE', Geosci. Model Dev., 4, 957-992, doi:10.5194/gmd-4-957-2011, 2011). The first configuration in each of the five blocks above corresponds to a 5-kyr spin-up configuration for the corresponding setup (labelled in the publication as '3636s16l', '"biogem"', '7272s32l', '6432i16l', and '3660l16l', repsectively), the other configurations are modified versions thereof (selecting alternative model-data evaluation of atmospheric fields, doubled simulation length, or alternative paths around Antarctica for the computation of the barotropic streamfunction).
