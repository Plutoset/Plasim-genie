gd_jobsubmit

Submits a compute job to a Globus GRAM job manager

Syntax

  • JOBHANDLE = gd_jobsubmit(RSL,HOST)

Description

JOBHANDLE = gd_jobsubmit(RSL,HOST) This command submits a compute job to a Globus server running a GRAM job manager. Where RSL is a string describing the submitted job, HOST is the name of the Globus server, and JOBHANDLE is the handle for a successfully submitted job.

Remarks

Note that a valid proxy certificate is required to submit a GRAM job. For more information about RSL see http://www.globus.org/gram/.

Examples

This is an example job submission.

  • jobHandle = gd_jobsubmit('&(executable =/bin/date)','myhost.mydomain.com')
