gd_proxyquery

Queries whether a valid proxy certificate exists.

Syntax

  • ISVALID = gd_proxyquery

  • ISVALID = gd_proxyquery(FLAG, NUMVALUE)

Description

ISVALID = gd_proxyquery This command determines whether a valid proxy certificate exists for user's certificate. where ISVALID is 1 if a valid proxy certificate exists at the location specified by the COG.PROPERTIES file, otherwise 0.

ISVALID = gd_proxyquery(FLAG, NUMVALUE) This queries the strength or time remaining during which the proxy certificate is valid. Where EXISTS is 1 if the proxy certificate meets the requirements of remaining lifetime or cryptographic strength, otherwise 0. If FLAG = 'time' the time remaining for the proxy certificate is queried against minValue hours. If FLAG = 'strength' the cryptographic strength of the proxy certificate is queried against MINVALUE bits.

Example

This example returns EXISTS = 0 for a proxy certificate of strength 512.

  • exists = gd_proxyquery('strength',1024)
