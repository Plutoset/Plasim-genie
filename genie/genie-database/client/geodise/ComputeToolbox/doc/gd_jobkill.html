gd_jobkill

Kills a Globus GRAM specified by job handle

Syntax

  • gd_jobkill(JOBHANDLE)

Description

gd_jobkill(JOBHANDLE) Kills the Globus job specified by the Globus job handle.

Remarks

A valid proxy certificate for the correct user credentials is required to kill a GRAM job.

Examples

This example terminates a Globus job using the job handle returned by gd_jobsubmit.

  • gd_jobkill('https://escience-dept2.sesnet.soton.ac.uk:33342/10288/1029768382/')
