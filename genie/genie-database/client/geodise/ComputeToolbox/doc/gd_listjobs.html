gd_listjobs

Returns all the job handles belonging to the user

Syntax

  • JOBHANDLES = gd_listjobs(HOST,PORT)

Description

JOBHANDLES = gd_listjobs(HOST,PORT) returns a cell array of Globus GRAM job handles for the current user's credentials following a query to the MDS server on the machine HOST on the specified PORT

Remarks

Note that a proxy certificate is required, as the queries are based upon the user's credentials.
