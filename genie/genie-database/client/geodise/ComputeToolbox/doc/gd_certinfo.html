gd_certinfo

Returns information about the user's certificate

Syntax

  • SUBJECT = gd_certinfo

  • SUBJECT = gd_certinfo(FILE)

Description

subject = gd_certinfo Prints details of the user's certificate configured for use by the Geodise Computational toolbox. Where SUBJECT is the certificate subject line in Globus format, suitable for use in a Globus gridmap file.

SUBJECT = gd_certinfo(FILE) where FILE is the filename of the certificate to be queried. The certificate must be encoded in pem format.

Remarks

The default location of the user's certificate is configured by the cog.properties file.
