gd_rmfile

Deletes a remote file using GridFtp

Syntax

  • gd_rmfile(HOST,FILE)

Description

gd_rmfile(HOST,FILE)Deletes the file specified by the string FILE on the GridFTP server specified by the string HOST.

Remarks

Note that a valid proxy certificate is required to use GridFTP. Suitable credentials will be required to delete a file on a GridFTP server.
