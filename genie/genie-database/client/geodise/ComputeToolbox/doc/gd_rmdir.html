gd_rmdir

Deletes a remote directory using GridFTP

Syntax

  • gd_rmdir(HOST,DIRECTORY)

Description

gd_rmdir(HOST,DIRECTORY) Deletes a directory specified by the string DIRECTORY on the GridFTP server specified by the string HOST.

Remarks

Note that a valid proxy certificate is required to use GridFTP. Suitable credentials will be required to delete a directory on a GridFTP server.
