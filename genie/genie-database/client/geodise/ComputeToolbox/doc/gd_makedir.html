gd_makedir

Creates a remote directory using GridFtp

Syntax

  • gd_makedir(HOST,DIRECTORY)

Description

gd_makedir(HOST,DIRECTORY) Creates a directory specified by the string DIRECTORY on the GridFTP server specified by the string HOST.

Remarks

Note that a valid proxy certificate is required to use GridFTP. Suitable credentials will be required to create a directory on a GridFTP server.
