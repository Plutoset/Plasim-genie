gd_createproxy

Creates a Globus proxy certificate for the user's credentials

Syntax

  • gd_createproxy

Description

gd_createproxy This command creates a Globus proxy certificate for the user's credentials at the location specified by the cog.properties file. The user is queried for the passphrase to their private key by a pop-up window.
