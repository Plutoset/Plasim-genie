gd_jobstatus

Gets the status of a Globus GRAM job

Syntax

  • STATUS = gd_jobstatus(JOBHANDLE)

Description

STATUS = gd_jobstatus(JOBHANDLE) returns the status of a Globus GRAM job, where status:

  • -1 is Unknown

  • 1 is PENDING

  • 2 is ACTIVE

  • 3 is DONE

  • 4 is FAILED

  • 5 is SUSPENDED

  • 6 is UNSUBMITTED

Example

This example queries the status of a Globus job using the job handle returned by gd_jobsubmit.

  • gd_jobstatus('https://escience-dept2.sesnet.soton.ac.uk:33342/10288/1029768382/')
